# X-Ray Bone Analyzer - Kullanım Talimatları

## Kurulum
1. Scriptleri çalıştırılabilir yapın:
```bash
chmod +x train.sh start.sh
```

## Eğitim Modu
Eğitim modunu başlatmak için:
```bash
./train.sh
```
Bu script:
- Sanal ortamı oluşturur (eğer yoksa)
- Gerekli paketleri yükler
- Model eğitimini başlatır
- Eğitim tamamlandığında bilgi verir

## GUI/Test Modu
GUI'yi başlatmak için:
```bash
./start.sh
```
Bu script:
- Sanal ortamı oluşturur (eğer yoksa)
- Gerekli paketleri yükler
- GUI'yi başlatır
- GUI kapandığında bilgi verir

## Ortak Özellikler
- Her iki script de otomatik olarak:
  - Sanal ortamı oluşturur ve aktifleştirir
  - Gerekli paketleri yükler
  - İşlem bittiğinde nasıl deaktive edileceğini hatırlatır

## Notlar
- İşlemler bittiğinde sanal ortamı deaktive etmek için:
  ```bash
  deactivate
  ```
- Scriptler herhangi bir platformda çalışır (macOS, Linux)
- Sistem ayarlarını değiştirmenize gerek yoktur
- Projeyi başka bir bilgisayara taşıdığınızda aynı şekilde çalışır
